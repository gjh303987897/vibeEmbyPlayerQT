# Emby / Jellyfin 页面级加载提示

## 背景

用户反馈：首次进入 Emby 主页面，或者进入某个媒体库时，如果遇到网络抖动，页面内容区可能短时间空白，只能看到顶部工具栏里的小加载图标。

这会让用户难以判断应用是否正在请求服务器数据。

本次为 Emby / Jellyfin 共用的服务首页和媒体库页面增加页面级加载提示。

## 阅读与复查

实现前复查了以下内容：

- `qml/Main.qml`
  - 服务首页对应 `home` 页面，包含继续观看和媒体库两个区域。
  - 媒体库内容页对应 `library` 页面，展示 `appViewModel.items`。
  - 顶部工具栏已有全局 `BusyIndicator`，但内容区没有首屏加载反馈。
- `src/viewmodels/AppViewModel.*`
  - `refreshHome()` 会并行刷新继续观看与媒体库列表。
  - `refreshLibraries()` 会使用全局 `loading`。
  - `refreshContinueWatching()` 原本没有独立加载状态。
  - `loadMediaDirectory()` 负责加载媒体库条目。
- 播放器相关历史修复记录
  - 本次只增加页面加载状态与 QML 提示，不修改 `PlayerController`、`MpvVideoItem`、mpv 参数、native window 嵌入或播放上报逻辑。

本次不涉及 Emby / Jellyfin API 路径、请求参数或响应解析，因此无需调整服务端 API 实现。

## 实现内容

### 1. 新增服务首页加载状态

`AppViewModel` 新增：

```cpp
Q_PROPERTY(bool homeLoading READ homeLoading NOTIFY homeLoadingChanged)
```

内部使用计数器记录首页相关请求：

- 继续观看列表请求。
- 媒体库列表请求。

这样首页两路请求并行时，只要其中任意一路未完成，`homeLoading` 就会保持为 `true`。

### 2. 新增媒体库条目加载状态

`AppViewModel` 新增：

```cpp
Q_PROPERTY(bool libraryItemsLoading READ libraryItemsLoading NOTIFY libraryItemsLoadingChanged)
```

`loadMediaDirectory()` 请求开始时设置为 `true`，请求结束或过期响应返回时设置为 `false`。

该状态只表示媒体库条目区域是否正在读取，不改变已有的全局 `loading` 行为。

### 3. 新增页面加载面板

QML 中新增 `PageLoadingPanel`：

- 半透明现代化面板。
- 居中 `BusyIndicator`。
- 支持标题和副标题。
- 深色 / 浅色主题下使用不同透明底色与边框。

服务首页在以下条件满足时显示：

```qml
appViewModel.homeLoading
    && appViewModel.continueItems.count === 0
    && appViewModel.libraries.count === 0
```

媒体库页面在以下条件满足时显示：

```qml
appViewModel.libraryItemsLoading && appViewModel.items.count === 0
```

因此：

- 首次进入页面且内容为空时，会显示明显的加载提示。
- 已有内容的刷新或分页加载不会被大面板遮挡。

### 4. 多语言文案

新增中英文文案：

- `loading.home`
- `loading.homeHint`
- `loading.library`
- `loading.libraryHint`

## 影响范围

本次改动影响：

- Emby / Jellyfin 服务首页首屏加载提示。
- Emby / Jellyfin 媒体库首屏加载提示。
- 首页继续观看请求的独立加载状态。
- 媒体库条目请求的独立加载状态。

不影响：

- Emby / Jellyfin API 路径。
- 媒体库和媒体项解析。
- 播放器、mpv、native window 嵌入。
- WebDAV / SMB / IPTV 逻辑。
- SQLite 数据结构。

## 验证

已执行：

```powershell
.\scripts\build-clang.cmd
```

本地构建通过，包含 QML cache 生成和 C++ 编译链接。

需要人工点测：

- 首次进入 Emby 主页面，内容未返回前显示页面级加载提示。
- 首次进入 Jellyfin 主页面，内容未返回前显示页面级加载提示。
- 进入 Emby 媒体库，条目未返回前显示页面级加载提示。
- 进入 Jellyfin 媒体库，条目未返回前显示页面级加载提示。
- 已有内容时刷新不会遮挡现有列表。
