# 原生标题栏外观优化

## 背景

应用使用系统原生窗口标题栏。深色主题下，Windows 默认标题栏仍可能显示为白色，与播放器整体深色界面割裂，顶部还能看到明显的白条和系统窗口按钮区域。

## 修改内容

- 新增 `WindowAppearanceController`，用于统一处理主窗口原生标题栏外观。
- Windows 平台下通过 DWM 属性设置：
  - 深色标题栏模式
  - 标题栏背景色
  - 窗口边框颜色
  - 标题文字颜色
- 在 `main.cpp` 中创建并暴露 `windowAppearanceController` 给 QML。
- 在 `Main.qml` 根窗口完成初始化后绑定窗口，并在主题变化时重新应用标题栏颜色。
- CMake 增加 `dwmapi` 链接，仅在 Windows 平台启用。

## 设计说明

本次没有改成无边框自绘标题栏，因为原生标题栏可以保留系统级拖拽、缩放、最小化、最大化和关闭行为，风险更低，也更符合最小改动原则。

非 Windows 平台当前保持 no-op，不改变 macOS/Linux 的系统窗口行为。

## 验证

- 已通过构建验证。
- 需要在 Windows 桌面运行应用，确认深色主题下顶部白条变为与应用背景一致的深色标题栏。
- 需要切换浅色/深色主题，确认标题栏颜色会跟随更新。
