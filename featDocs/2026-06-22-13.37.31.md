# WebDAV 支持第一版

## 需求

- 服务添加对话框新增 `WebDAV` 类型。
- WebDAV 服务需要填写服务名称、完整 Endpoint、用户名、密码、自动登录、允许自签名证书。
- 保存后服务页显示 WebDAV 服务卡片。
- 点击 WebDAV 卡片后浏览远程文件/文件夹。
- 点击视频文件直接播放。
- 支持文件/文件夹上传。
- 支持文件/文件夹下载。
- 设置页支持配置默认下载目录；未配置时下载前弹出目录选择器。
- 下载前检查磁盘空间；空间不足或无法确认大小时弹警告，但允许继续。
- 上传/下载显示后台传输任务、进度和取消按钮。
- WebDAV 密码不写入 SQLite，自动登录依赖系统凭据库；不可用时降级为每次输入密码。

## 实现内容

### 服务类型

- 在 `ServiceType` 中新增 `WebDAV`。
- 服务添加对话框类型下拉增加 `WebDAV`。
- 保存 WebDAV 服务时复用 `servers` 表，仅保存 Endpoint、用户名、自动登录、自签名证书等非敏感信息。
- WebDAV 服务卡片点击后优先从系统凭据库读取密码；不可用或没有密码时弹出密码输入框。

### 凭据存储

- 新增 `CredentialStore`。
- Windows 使用 Credential Manager 保存 WebDAV 密码。
- 非 Windows 平台当前返回不可用，自动降级为点击服务卡片时输入密码。
- 删除 WebDAV 服务且选择删除本地数据时，会删除对应系统凭据。

### WebDAV 协议

- 新增 `WebDavClient`。
- 使用 `PROPFIND + Depth: 1` 浏览当前目录。
- 使用 `PROPFIND + Depth: 0` 获取单项信息。
- 使用 `MKCOL` 创建远程目录。
- 认证通过 `QNetworkAccessManager::authenticationRequired` 与 `QAuthenticator` 完成，支持 Qt 原生 Basic/Digest 认证。
- 自签名证书确认复用现有证书弹窗流程。

### 文件浏览与播放

- 新增 `WebDavItem` 与 `WebDavItemListModel`。
- WebDAV 页面显示当前目录、文件夹、文件、视频标记。
- 点击文件夹进入目录。
- 点击常见视频文件直接播放。
- 播放 WebDAV 视频时，`MpvVideoItem` 与 `PlayerController` 支持传入临时 HTTP 用户名/密码给 libmpv，不写日志。

### 上传/下载与后台任务

- 新增 `TransferTask`、`TransferTaskListModel`、`TransferManager`。
- 支持后台单队列上传、下载、远程建目录。
- 上传文件夹时保留目录结构，并把远程目录创建纳入队列，避免目录未创建就 PUT 文件。
- 下载文件夹时递归展开目录并保留结构。
- 上传/下载任务显示状态、进度、详情和取消按钮。
- 传输完成后如果当前在 WebDAV 页面，会刷新当前目录。

### 下载目录与空间检查

- `SessionRepository` 新增默认下载目录设置读写。
- 设置页新增 WebDAV 默认下载目录配置。
- 下载文件时根据远程大小和本地磁盘可用空间做检查。
- 下载文件夹前递归估算总大小；若无法确认全部大小，则弹出“无法确认大小”警告，用户可继续。
- 本地重名文件或文件夹按 `name (1).ext` 规则自动重命名。

## 验证

- 已查阅 WebDAV RFC 4918 与 Qt 网络认证、存储空间相关官方文档。
- 已执行 `.\scripts\build-clang.cmd`，构建通过。
- 已启动 `build-clang\vibePlayerQT.exe` 做冒烟验证，程序启动 6 秒后仍正常运行。
